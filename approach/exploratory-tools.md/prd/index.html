    Prediction-Realization Diagram

    The function plotPRD of the forvision package plots the Prediction-Realization Diagram for the given data from dataset containing columns value, forecast, and method (AFTS).

    Here’s a prediction-realization diagram created by the function plotPRD:

    # Load time series and forecast data from url
ts <- read.csv("https://forvis.github.io/data/M3_yearly_TSTS.csv", stringsAsFactors = FALSE)
fc <- read.csv("https://forvis.github.io/data/M3_yearly_FTS.csv", stringsAsFactors = FALSE)
# Create AFTS shema
library(forvision)
af <- createAFTS(ts, fc)
plotPRD(af)

    The resulting prediction-relization digram plotted in logarithmic scales by using the option setting useLog = TRUE:

    library(forvision)
plotPRD(af, useLogs = TRUE)

    The prediction-realization diagram for the selected methods:

    # For the methods "ARARMA", "Auto-ANN", and "DAMPEN"
data1 <- subset(af, method == c("ARARMA", "Auto-ANN", "DAMPEN"))
library(forvision)
plotPRD(data1)

    The prediction-realization diagram for the selected horizon:

    # For the horizons 1, 2
data2 <- subset(af, horizon == 1)
library(forvision)
plotPRD(data2)

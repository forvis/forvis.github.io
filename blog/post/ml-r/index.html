Giới thiệu một số Machine Learning Datasets trên R

November 13, 2018
R, Machine Learning
Page content
        • 1. Package dataset
        • 2. Package: mlbench

Ở post này tôi sẽ giới thiệu một số Machine Learning Datasets trên R.

Datasets là nguyên liệu vô cùng quan trọng giúp chúng ta có thể test, thực hành và thử nghiệm với các machine learning techniques và phát triển kỹ năng của mình với platform. Ở mục này các bạn sẽ được giới thiệu các cách để load các standard datasets dành cho bài toán classification và regression trên R. Cụ thể là các bạn sẽ biết được:

  • Các R packages mà các bạn có thể dùng để load standard machine learning datasets.

  • Các datasets cụ thể mà bạn có thể sử dụng để thực hành machine learning.

1. Package dataset

Đầy là gói cơ sở đi kèm với R, điều này có nghĩa là các bạn không cần phải load package này. Nó bao gồm một số lượng lớn datasets mà bạn có thể sử dụng. Để load một dataset từ package này các bạn gõ lệnh sau:

data(TenCuaDataset)

Để kiểm tra danh sách tất cả các datasets có trong package các bạn gõ:

# list all datasets in the package
library(help = "datasets")

Dưới đây là một số datasets có trong package này:

  • UCI Machine Learning Repository:https://archive.ics.uci.edu/ml/datasets/Iris.

2. Package: mlbench

Mô tả package: Tập hợp (bộ sưu tập) các bài toán nhân tạo và thực tế về machine learning, bao gồm một số datasets từ UCI repository. Để tim hiểu thêm về mlbench package các bạn có thể tìm kiểm thông tin trên trang CRAN.

Nếu chưa cài mlbench package các bạn có thể cài đặt bằng cách gõ lệnh sau:

install.packages("mlbench")

Và load package bằng cách:

# load the package
library(mlbench)

Để kiểm tra danh sách tất cả các datasets của package này:

# list the contents of the package
library(help = "mlbench")

Dưới đây là một vài datasets từ package này:

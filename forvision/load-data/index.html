    Loading data

    Read data from URL (from example datasets)

    You can use the following methods to read data from URL sources:

    1) Using function download.file():

    temporaryFile <- tempfile()
download.file("RUL", destfile=temporaryFile)
myData <- read.csv(temporaryFile)

    2) Using package RCurl:

    library(RCurl)
myCsv <- getURL("RUL", ssl.verifypeer = FALSE)
myData <- read.csv(textConnection(myCsv))

    3) Using R base function read.csv():

    myData <- read.csv("RUL")

    You can use the RULs for examples datasets from the section datasets

    For example, to read dataset example1_AFTS.csv from URL:

    data <- read.csv("https://raw.githubusercontent.com/forvis/forvision_data/master/example1_AFTS.csv")
